*Where you can say "Hello World" in almost every programing language*

High level programing languages

- Python : 
print("Hello World")

- Lua : 
print("Hello World")

- Javascript : 
console.log("Hello World")

- Rust : 
fn main() {
    println!("Hello World!");
}

- Ruby :
puts "Hello World"

- Java :
import java.util.*;

public class Main {
    public static void main(String[] args) {
      System.out.println("Hello, World!");
    }
}

Low Level programing languages

- C++ : 
#include <iostream>
int main() {
    std::cout << "Hello World";
    return 0;
}

- C :
#include <stdio.h> 
int main() {
    printf("Hello World");
    return 0;
}

using System;

namespace HelloWorld
{
	public class Program

		public static void Main(string[] args)

		}
	}
